## What is Claude Craft?

A collection of **bilingual (FR/EN) copy-paste prompt templates** for [Claude Code](https://claude.ai/code), leveraging skills, MCP servers (fontofweb, context7), and agent swarms.

## How to use

1. Browse the categories in the sidebar
2. Find the prompt template you need
3. Switch between **FR** and **EN** tabs
4. Click the **copy button** on the code block
5. Paste into Claude Code and replace the `@path` placeholders with your actual paths

## Categories

| Category | Description |
|----------|-------------|
| [PRD & Planning](/docs/prd-planning) | Generate and review Product Requirements Documents |
| [Code Quality](/docs/code-quality) | Optimize, refactor, audit, and debug your code |
| [Design](/docs/design) | Review and improve component design with inspiration |
| [DevOps & Migration](/docs/devops) | Build, deploy, rename, and migrate your stack |
| [Skills](/docs/skills) | Create and update Claude Code skills |

## Conventions

- **`@path`** references (e.g., `@src/components/`, `@tasks/prd-name.md`) are placeholders — replace them with your actual file/folder paths
- **`/skill-name`** references indicate which Claude Code skills the prompt relies on
- **MCP servers** (fontofweb, context7) are referenced when the prompt needs external data
- **PRD prompts** always output to a `@tasks/` folder using `/ralph-tui-prd`
