# Create Skill

Créer un nouveau skill Claude Code en utilisant /skill-creator et context7 pour maîtriser parfaitement le sujet du skill.

## FR

```
Crée un nouveau skill pour tanstack-router en utilisant le skill /skill-creator et context7 pour apprendre parfaitement tanstack-router. Sauvegarde le skill dans le dossier ~/.claude/skills/, puis également dans le dossier /home/arthur/dev/skills pour le partager à ma communauté. Une fois fait, commit et push les changements en respectant conventional commit.
```
