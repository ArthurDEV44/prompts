# Refactor Code

Refactorer du code existant pour améliorer la structure, la lisibilité et la maintenabilité : extraire des fonctions, séparer les responsabilités, appliquer des design patterns, réduire la duplication. Contrairement à optimize-code qui se concentre sur la performance, ce prompt se concentre sur l'architecture et les principes de clean code. Utilise context7 et les skills pertinents de la bibliothèque selon le code refactoré.

## Single file

Refactorer un seul fichier ciblé.

### FR

```
Explore le fichier @backend/api/sources.rs, puis refactore le code : extrais des fonctions, sépare les responsabilités, réduis la duplication, et applique les design patterns appropriés. Utilise context7 pour approfondir ta compréhension de la stack technique, et utilise les skills pertinents de ta bibliothèque si applicables au code refactoré.
```
